# 奋隼建筑官网与经验分享平台整合方案

## 官网现有内容分析

### 页面结构
1. **首页 (Hero)** - 公司品牌展示
2. **业务领域 (Services)** - 房屋建筑、市政工程、装饰装修
3. **工程案例 (Projects)** - 代表性项目展示
4. **公司简介 (About)** - 公司介绍和统计数据
5. **联系我们 (Contact)** - 联系方式

### 设计风格
- 简洁专业的白色背景
- 黑色为主色调
- Helvetica Neue 字体
- 卡片式布局
- 悬浮动效

## 整合后的网站结构

### 导航结构
```
首页
├── 英雄区 (Hero)
├── 业务领域 (Services)
├── 工程案例 (Projects)
├── 公司简介 (About)
├── 经验分享 (预览 + 链接)
└── 联系我们 (Contact)

经验分享 (/stories)
├── 文章列表
├── 分类筛选
└── 搜索功能

文章详情 (/stories/:slug)
└── Markdown 内容渲染
```

### 设计风格统一
- 保持官网的简洁专业风格
- 使用白色背景 + 黑色主色调
- 添加橙色作为强调色（与原经验分享网站呼应）
- 统一字体和间距系统

### 技术实现
- React + TypeScript
- Tailwind CSS
- Framer Motion 动画
- Markdown 文章加载
- 响应式设计
